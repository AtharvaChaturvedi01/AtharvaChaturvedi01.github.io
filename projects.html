Projects

F1 2021 Season Analysis

Conducted in-depth analysis of the Formula 1 2021 season using Python and FastF1. Employed web scraping techniques with BeautifulSoup and Pandas to gather race results, schedules, and timing data. Utilized Matplotlib and Seaborn for visualizations, showcasing the final points table, driver performance trends, and dynamic insights from the Abu Dhabi Grand Prix. Demonstrated proficiency in data gathering, analysis, and visualization for impactful insights in a dynamic sports environment.

Tic-Tac-Toe

Created a web-based implementation of the game in Java allowing players to take turns and make valid moves on a grid-based board. Implemented player input handling, game board updating, and win determination by checking rows, columns, and diagonals for matching symbols, also allowed users to create an account and sign-up, and had a win-based leaderboard.

3d Shooting Game

Created a third person shooting game using Unity allowing players to move in all directions, shoot, collect ammo, and heal up. Game included an opening menu, pause menu, and had a leaderboard based on the highest number of kills.

Airport Simulator

Created a Java program that logs all flights arriving and departing from an airport, using object-oriented concepts and efficient data structures. Offers a user-friendly interface to add, remove, and view flight details, while validating required fields and times.
